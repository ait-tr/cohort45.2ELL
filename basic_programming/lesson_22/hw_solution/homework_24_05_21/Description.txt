Определите временную сложность данных алгоритмов и объясните свой выбор!




Ответы:

## 1 Общая сложность - константная сложность O(1)
    По тому что, временная сложность практически не зависит
    от нескольких операций ввода / вывода информации.

## 2 Общая сложность - константная сложность O(1)
    По тому что,  временная сложность не находится
    в зависимости от условных конструкций.

## 3 Общая сложность - константная сложность O(1)
    По тому что, во всех этих трёх циклах количество итераций известно заранее.

## 4 Общая сложность - в четвёртой степени O(n^4)
    По тому что, выражение "i < n * n" во внешнем цикле по сути означает ещё один вложенный цикл
    и выраженме "j < n * n" во внутреннем цикле то же означает ещё один вложенный цикл,
    в итоге получается четвёртая степень и циклы будут выполнятся 100^4 раз, практически бесконечно,
    что не есть хорошо!

## 5 Общая сложность - квадратичная временная сложность O(n^2)
     По тому что, выражение "i < n * n" в данном цикле означает ещё один вложенный цикл.

## 6 Общая сложность - константная сложность O(1)
    По тому что, значение "n * n * 0" равно нулю, следовательно цикл не выполнится ни разу.
